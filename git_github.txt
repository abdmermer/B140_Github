--Localimizde repo olusturabilmek icin
1-Calisacagimiz klasoru secip yada yeni bir klasor olusturup,ilgili yere cmd yazip 
terminali acacagiz.
2-dizine git init yazariz.
3-calisma alanina working space denir.
4-git add komutuyla staging area alanina tasiriz
5-git commit -m'version adi' ile yazarak version olustururuz
6-git status ile son durumu goruruz.
7-git log --oneline ile versionlari goruruz.
8-versionu github'a gondermek icin git push <github repository adresi> komutu/ 
/git remote add origin<repo adresi>  / git push -u master
9-githubdaan veri cekebilmek icin git pull <repo adresi>
